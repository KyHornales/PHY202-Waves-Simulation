# Wave Simulation Lab

An interactive browser-based physics simulation for visualizing wave behavior through real-time controls, dynamic animation, and concept-based learning.

This project was developed as an educational tool for studying core wave mechanics in a more intuitive and engaging format.

---

# Overview

Wave Simulation Lab allows users to explore the relationships between wave properties such as amplitude, frequency, wavelength, and wave speed through direct interaction.

Rather than relying only on static diagrams or equations, the simulator demonstrates how parameter changes affect wave motion instantly.

This project is suitable for:

* Students studying introductory physics
* Teachers and presenters
* STEM demonstrations
* Educational web projects
* Self-guided learning

---

# Key Features

## Simulation Modes

### Single Wave Mode

Visualize a single traveling wave and modify its characteristics in real time.

### Interference Mode

Observe how two waves interact to produce:

* Constructive interference
* Destructive interference

### Standing Wave Mode

Demonstrate stationary wave patterns formed by two identical waves traveling in opposite directions.

Includes:

* Nodes
* Antinodes

---

# Interactive Controls

Users can adjust wave properties through responsive controls:

* Amplitude
* Frequency
* Wavelength
* Speed (if enabled)

All changes are reflected immediately in the simulation.

# Educational Value

The goal of this simulator is to connect mathematical equations with observable motion.

For example:

`v = fλ`

Users can adjust frequency and wavelength, then immediately observe corresponding changes in wave speed.
